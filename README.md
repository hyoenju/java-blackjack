# 연료 주입

## 기능요구사항

- [ ] 렌터카는 Sonata 2대, Avante 1대, K5 2대로 총 5대의 차량을 보유하고 있다.
    - 차량 별 연비 정보를 저장하고 있다.
      - Sonata : 10km/리터
      - Avante : 15km/리터
      - K5 : 13km/리터)
    - 차량 별 보유 대수를 저장하고 있다.
- [ ] 고객은 여행할 목적지의 대략적인 이동거리를 입력 받는다.
- [ ] 이 이동거리를 활용해 차량 별로 필요한 연료를 주입한다.
- [ ] 차량 별로 주입해야 할 연료량을 확인할 수 있는 보고서를 생성한다.
